Hi there! 👋 I'm a developer who loves building smart, user-friendly applications—and occasionally diving into game mechanics.

When I'm not writing code, you can usually find me gaming (I even built my own Flappy Bird clone in Unity with custom physics) or tinkering with hardware and microcontrollers. Lately, I have been deep-diving into AI engineering, building out RAG pipelines, and shipping full-stack applications with the FARM stack. 

### Some of my projects

* **Doc Formatter:** Built an AI-powered SaaS that uses a hybrid ML pipeline and the Google Gemini API to automatically format academic lab reports with 95%+ accuracy.
* **AI Casino:** Engineered a browser-based interactive casino game featuring a real-time AI dealer and on-demand image generation, adjusting the gameplay odds dynamically based on how the user plays.
* **AttentionOS:** Designed a privacy-first Android app that tracks focus and screen time entirely on-device, meaning no personal data ever hits a server.
* **Flappy Bird Clone (Unity 2D):** Independently built a complete game in Unity (C#) featuring custom physics, procedural pipe spawning, and a persistent scoring system.
